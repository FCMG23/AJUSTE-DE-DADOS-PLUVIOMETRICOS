AJUSTE DE DADOS PLUVIOMÉTRICOS

Esta ferramenta foi desenvolvida para auxiliar no tratamento de dados pluviométricos disponibilizados pelo CEMADEN, especialmente os obtidos no Mapa Interativo (https://mapainterativo.cemaden.gov.br/
). Os dados são originalmente fornecidos em arquivos CSV com registros em escala horária, o que exige processamento para análises climatológicas em diferentes escalas temporais.

A aplicação permite automatizar o processo de agregação dos dados, gerando somatórios de precipitação em bases diária, semanal e mensal, facilitando o uso em análises geográficas e climatológicas.

 AJUSTE DE DADOS PLUVIOMÉTRICOS – Multi

A versão Multi permite o processamento em lote de diversos arquivos CSV simultaneamente.

Funcionamento:
O usuário seleciona a pasta onde estão armazenados os arquivos CSV baixados.
Em seguida, define a pasta de destino onde serão salvos os resultados processados.
Ao executar, a ferramenta realiza automaticamente o somatório dos dados e gera novos arquivos com agregações:
Diária
Semanal
Mensal

Caso ocorra algum erro durante o processo ou ao final da execução, a aplicação exibe mensagens informativas de status (sucesso ou falha).

AJUSTE DE DADOS PLUVIOMÉTRICOS – Single

A versão Single permite o processamento individual de um único arquivo CSV.

Funcionamento:
O usuário seleciona o arquivo CSV desejado.
Define a pasta onde os resultados serão salvos.
Executa o processamento para geração dos dados agregados (diário, semanal e mensal).

Observação
A ferramenta foi pensada para otimizar fluxos de trabalho em análises hidroclimáticas, reduzindo o tempo gasto com manipulação manual de dados horários e permitindo maior foco na interpretação geográfica e climatológica dos resultados.
